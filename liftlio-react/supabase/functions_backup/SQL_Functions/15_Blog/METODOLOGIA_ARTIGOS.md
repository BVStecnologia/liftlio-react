## Etapa 1: Pesquisa Google Trends

**Objetivo:** Identificar o que as pessoas estao buscando AGORA

**Ferramentas:**
- Google Trends (trends.google.com)
- Think with Google (thinkwithgoogle.com)
- YouTube Trends (para topicos video-related)

**O que buscar:**
- Tendencias em ascensao (rising)
- Sazonalidade de termos
- Comparacao entre termos similares
- Queries relacionadas

**Exemplo de pesquisa:**
```
Termo: "youtube marketing"
- Verificar: esta subindo ou caindo?
- Queries relacionadas: "youtube comment marketing", "ai youtube"
- Geografias: US, UK, BR
```

## Etapa 2: Keywords e Intencao de Busca

**Tipos de Intencao:**

| Intencao | Exemplo | Conversao |
|----------|---------|-----------|
| Informacional | "what is comment marketing" | Baixa |
| Comparativa | "tool A vs tool B" | ALTA |
| Transacional | "best youtube marketing tool" | ALTA |
| Navegacional | "liftlio login" | N/A |

**Priorizar:**
1. Comparativos (10x mais conversao)
2. Listicles com numeros impares (7, 11, 15)
3. How-tos com resultado especifico
4. Objecoes comuns ("is X spam?", "does X work?")

**Keywords para Liftlio:**
- "youtube comment marketing"
- "ai youtube engagement"
- "brand monitoring youtube"
- "first comment youtube strategy"
- "youtube lead generation comments"

## Etapa 3: Estrutura do Artigo

### Tipos que Funcionam

| Tipo | Trafego | Conversao | Exemplo |
|------|---------|-----------|---------|
| Listicle | 2x | Media | "7 Mistakes That Kill..." |
| Comparativo | 1x | 10x | "Best Tools 2026 (Honest)" |
| How-To | 1.5x | Alta | "How to Set Up in 15 Min" |
| Objecao | 1x | Alta | "Is It Spam? Honest Answer" |
| Case Study | 0.5x | MUITO Alta | "How X Got 47 Leads" |
| Pillar | 3x | Media | "Complete Guide to..." |

### Estrutura Padrao

```markdown
# Titulo com Numero ou Beneficio Claro

## Introducao (Hook + Problema)
- Estatistica impactante
- Pergunta provocativa
- Historia relatable

## Corpo (80% Educacao Pura)
- Subsecoes claras (H2, H3)
- Exemplos praticos
- Dados e pesquisas
- Screenshots quando aplicavel

## Transicao Natural (Bridge)
- "Given these challenges..."
- "This is where automation helps..."
- "Smart marketers are solving this by..."

## Mencao Liftlio (Sutil)
- Nunca "Liftlio is the best"
- Sempre "Tools like Liftlio..."
- Ou "Platforms that monitor..."

## Conclusao + CTA
- Resumo do valor entregue
- CTA natural no final
- Nunca forcado
```

## Etapa 4: Redacao - Regra 80/20

### 80% Educacao Pura

O leitor deve aprender algo MESMO SEM usar Liftlio.

**Fazer:**
- Entregar valor real e acionavel
- Usar dados e pesquisas
- Dar exemplos concretos
- Resolver o problema do leitor

**Evitar:**
- Conteudo raso so pra rankear
- Promessas exageradas
- Clickbait sem substancia

### 20% Contexto do Produto

**Tecnica "Grendene" - Seducao Sutil:**

```
NAO FAZER:
"Use Liftlio! It's the best tool for YouTube marketing!"

FAZER:
"Manually monitoring thousands of comments is humanly impossible.
This is why AI-powered platforms have emerged - they scan conversations
24/7, identify buying intent, and alert you in real-time.

When evaluating such tools, look for:
- Real-time monitoring (not daily digests)
- Intent classification (PICS framework)
- Multi-channel coverage

[Aqui o leitor JA QUER saber qual ferramenta faz isso]
"

```

### Framework PAS (Problem-Agitate-Solution)

```
PROBLEM:  "You're spending $50/click on ads that get skipped"

AGITATE:  "Meanwhile, competitors are building organic presence
          in conversations where trust is already established.
          Every day you wait, they're capturing leads you'll
          never even know existed."

SOLUTION: "There's a third path..." → (introduce concept naturally)
```

## Politicas Google - O Que EVITAR

### Conteudo Proibido

| Categoria | Exemplo | Risco |
|-----------|---------|-------|
| Spam | Keyword stuffing | Penalizacao |
| Enganoso | Promessas falsas | Remocao |
| Clickbait | Titulo nao entrega | Bounce alto |
| Thin Content | <300 palavras | Nao rankeia |
| Duplicado | Copiar outros | Penalizacao |

### Boas Praticas E-E-A-T

**Experience:** Mostrar que usamos o que ensinamos
**Expertise:** Dados, pesquisas, profundidade
**Authoritativeness:** Citar fontes confiaveis
**Trustworthiness:** Ser honesto sobre limitacoes

### Regras de Ouro

1. **Nunca mentir sobre resultados**
   - "Some users see..." nao "You WILL get..."

2. **Nunca atacar concorrentes diretamente**
   - "Unlike some tools..." nao "Tool X is terrible..."

3. **Sempre citar fontes**
   - Links para estudos, dados oficiais

4. **Ser honesto sobre limitacoes**
   - "Liftlio works best for B2B SaaS..."
   - "If you're just starting, manual might be enough..."

5. **Conteudo original**
   - Nunca copiar, sempre criar

## Geracao de Imagens (OBRIGATORIO: Gemini)

### Ferramenta Oficial: Google Gemini

**⚠️ NUNCA usar Pollinations ou outras ferramentas gratuitas!**
O usuario tem assinatura do Gemini Pro - usar SEMPRE.

**Como gerar:**
1. Acessar gemini.google.com via MCP Browser
2. Login com valdair3d@gmail.com
3. Prompt em ingles com especificacoes claras
4. Baixar imagem em tamanho original
5. Upload para Supabase Storage (bucket: blog-images)
6. Atualizar cover_image_url no banco

**Exemplo de prompt:**
```
Generate an image: Professional marketing dashboard with
purple gradient background, modern UI design, clean aesthetic,
16:9 horizontal format for blog article about [TOPIC]
```

**Padroes de qualidade:**
- Formato: 16:9 horizontal
- Tema: Purple gradient (cores Liftlio)
- Estilo: Profissional, moderno, limpo
- Sem texto na imagem (exceto se necessario)

**Storage:**
- Bucket: `blog-images` (public)
- URL: `https://suqjifkhmekcdflwowiw.supabase.co/storage/v1/object/public/blog-images/[filename]`

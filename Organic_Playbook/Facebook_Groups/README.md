## Estratégia

1. **Lurk primeiro** (1-2 semanas) para entender cultura
2. **Responder perguntas** antes de promover
3. **Seguir regras religiosamente** - admins banem rápido
4. **Construir relação com admins**
5. **Nunca dropar links sem contexto**

## Estratégia Principal: Sindicalização

Republicar artigos do blog Liftlio com:
- **Canonical URL apontando para original** (OBRIGATÓRIO!)
- Modificações para a plataforma
- Cross-post 2 semanas após publicação original

## Canonical URL (CRÍTICO!)

**SEMPRE** setar canonical para o artigo original do blog:

### Medium
Adicionar em story settings

### Dev.to
```markdown
---
canonical_url: https://liftlio.com/blog/original-article
---
```

---

## Medium Best Practices

### Para Publicações
1. Encontrar publicações do nicho (10K+ followers)
2. Ler guidelines de submissão
3. Seguir editors, engajar com content
4. Pitch ideias específicas
5. Começar com publicações menores

### Writing Style
- Hook nas primeiras 2 frases
- Subheadings a cada 300-400 palavras
- Imagens quebram o texto
- 5-8 minute read optimal

---

## Dev.to Best Practices

### Tags que Funcionam
- #beginners - High traffic
- #tutorial - Discovery boost
- #webdev - Large community
- #productivity - Cross-over audience

### Post Format
```markdown
---
title: Your Title Here
published: true
description: 2-3 sentence description
tags: tag1, tag2, tag3
canonical_url: https://liftlio.com/blog/original
cover_image: https://url-to-image.jpg
---

Content here...
```

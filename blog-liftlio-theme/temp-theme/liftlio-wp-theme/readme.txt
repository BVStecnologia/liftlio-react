Clean and professional blog theme for Liftlio.

== Description ==
Liftlio Blog Theme is a modern, clean, and professional WordPress theme designed specifically for the Liftlio platform. It features a minimalist design with gradient accents, optimized for readability and performance.

== Features ==
* Clean, modern design
* Fully responsive
* Custom navigation walker
* Gradient brand colors
* Optimized for performance
* Translation ready
* Custom post meta (reading time)
* SEO friendly

== Installation ==
1. Upload the theme folder to the /wp-content/themes/ directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Configure your menus and widgets
